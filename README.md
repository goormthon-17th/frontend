## 🍊 핵심 기술 및 아키텍처 Architecture

### Front-end
| 기술명 | 선정 이유 |
| :--- | :--- |
| **Next.js** | 서버 사이드 렌더링(SSR)과 정적 생성(SSG)을 활용하여 빠른 로딩 속도와 SEO 최적화를 동시에 확보했습니다. |
| **TypeScript** | 정적 타입 기반으로 코드 안정성을 높이고, 협업 시 발생할 수 있는 오류를 사전에 방지했습니다. |
| **Vapor UI** | 디자인 시스템을 기반으로 UI 일관성을 유지하고, 컴포넌트 재사용성을 통해 개발 생산성을 향상시켰습니다. |

### Back-end
| 기술명 | 선정 이유 |
| :--- | :--- |
| **Express** | 경량화된 서버 프레임워크로 빠른 API 개발과 유연한 서버 구성이 가능하여 MVP 개발에 적합했습니다. |
| **MySQL** | 관계형 데이터베이스를 통해 데이터의 무결성과 안정적인 관리가 가능하도록 설계했습니다. |

### Infra & DevOps
| 기술명 | 선정 이유 |
| :--- | :--- |
| **Docker** | 애플리케이션을 컨테이너화하여 개발 및 배포 환경의 일관성을 유지했습니다. |
| **Kubernetes (K8s)** | 컨테이너 오케스트레이션을 통해 서비스의 확장성과 안정적인 운영 환경을 구축했습니다. |
| **Jenkins** | CI 파이프라인을 구축하여 코드 빌드 및 배포 과정을 자동화했습니다. |
